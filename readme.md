## Recomendaciones
* Utilizar un ambiente virtual de python
    * Crear el ambiente virtual
        * ```python -m venv .venv```
    * Activar el ambiente
        * ```./.venv/scripts/activate```
    * Instalar PLY (Python-Lex-Yacc)
        * ```pip install ply```
